    Curl介绍

    cURL 是一个利用 URL 语法在命令行下工作的文件传输工具。借助 cURL，使用 REST API 、基本认证进行存储管理，包括列目录、文件上下传、目录创建删除、文件删除等。

    Curl安装

    CURL安装方式有多种，配置环境变量和文件直接拷贝比较常见。这里只介绍最简单直接的一种，直接拷贝。

    下载后将压缩包解压到C，如图：

    进入curl7653目录，找到红框位置C:\curl-7.65.3\src文件进入，复制curl.exe

    将curl.exe拷贝到C:\Windows\System32

    使用

    测试curl命令，cmd输入命令 curl –help
    出现如图英文提示表示安装成功！

    下载

    我是2019年11月17日在官网下载的最新版本，已经上传到了百度云。提取码 835c

    curl-7.65.3下载

    其他方法

    使用方式二：
    放置在system32中
    解压下载好的文件，拷贝I386/curl.exe文件到C:\Windows\System32
    然后就可以在DOS窗口中任意位置，使用curl命令了。

    使用方式三：
    配置环境变量
    在系统高级环境变量中，配置
    CURL_HOME —– “你的curl目录位置\curl-7.43.0”
    path —- 末尾添加 “;%CURL_HOME%\I386”
    这样与上面方式二的效果相同。
